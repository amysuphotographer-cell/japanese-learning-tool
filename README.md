# 日语动词变形学习工具 🇯🇵

一个现代化、交互式的日语动词变形学习工具，帮助你轻松掌握日语动词的各种变形规则。

## ✨ 功能特点

### 📚 学习模式
- **11种动词变形**：ます形、て形、た形、ない形、可能形、意志形、命令形、ば形、被动形、使役形、使役被动形
- **自动类型识别**：智能识别一段动词、五段动词和不规则动词
- **假名标注**：每个变形都配有清晰的平假名标注，方便自学
- **罗马音显示**：帮助初学者正确发音
- **详细规则说明**：每种动词类型都有详细的变形规则解释
- **🔊 语音发音**：点击即可听到优化后的日语发音

### 🎯 练习模式
- **随机出题**：从常用动词库随机抽取题目
- **即时反馈**：答题后立即显示对错，并给出详细解释
- **统计功能**：实时显示正确率、答题数、连击数
- **假名提示**：答错时显示正确答案的假名标注

### 💡 内置词库
- **100+ 常用动词**：包含中文释义
- **三大类动词**：
  - 一段动词（る动词）：食べる、見る、起きる...
  - 五段动词（う动词）：書く、読む、話す...
  - 不规则动词：する、来る及各种复合动词

## 📝 变形类型说明

### 一段动词（る动词）
去掉「る」，加上相应词尾
- 例：食べる → 食べます、食べて、食べた...

### 五段动词（う动词）
词尾按五十音图的段变化
- 例：書く → 書きます、書いて、書いた...

### 不规则动词
特殊记忆
- する → します、して、した...
- 来る → 来ます、来て、来た...
